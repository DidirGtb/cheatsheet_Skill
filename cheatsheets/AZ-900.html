AZ-900 – Fondamentaux du Cloud

☁️ C’est quoi le Cloud ?

  • Machine (Hardware) qui n’est pas en notre possession.
  • On peut louer les capacités.
  • On ne paye que ce que l’on utilise.

🔒 Modèle de partage des responsabilités

Il définit la manière dont Microsoft et le client se partagent la responsabilité en matière de sécurité.

Infra On Premise

Toute la responsabilité incombe au client, car tout est chez lui.

Avantage du Cloud

Microsoft peut assurer une partie de la sécurité, notamment la partie physique des machines.

Deux modèles de responsabilité

  • Mixte : Microsoft assure la sécurité physique des machines, mais pas la partie software.
  • Mixte / Partagé :
    • Physique + OS → Microsoft
    • Réseau + Paramètres App → Partagé
    • Données + Utilisateurs → Client

🏗️ Cloud Privé, Public et Hybride

  • Cloud Privé : dédié à une seule organisation, plus de contrôle.
  • Cloud Public : accessible à plusieurs organisations via Internet, fourni par Microsoft Azure, AWS, GCP…
  • Cloud Hybride : combinaison des deux (privé + public).
