Linux – Commandes de base

Quelques commandes essentielles pour naviguer et gérer fichiers/dossiers sous Linux. Astuce : ajoute --help après une commande pour voir toutes ses options.

🌳 Navigation

  • pwd → Affiche le chemin du dossier courant
  • cd /mnt/c → Aller à la racine du disque C:
  • cd nom_dossier → Aller dans un dossier
  • cd .. → Remonter d’un niveau
  • cd ~ ou cd $HOME → Revenir au dossier personnel

📂 Lister le contenu

  • ls → Lister fichiers/dossiers
  • ls -A → Inclure fichiers cachés
  • ls -l → Avec détails (droits, taille, date, propriétaire)
  • ls -rtl → Tri par date (les plus récents en dernier)

📄 Lire un fichier

  • more fichier.txt → Affiche page par page
  • less fichier.txt → Plus flexible que more

🛠️ Gérer fichiers & dossiers

  • mkdir mon_dossier → Créer un dossier
  • mv fichier.txt mon_dossier/ → Déplacer un fichier ⚠️ Si le dossier n’existe pas, mv renomme le fichier
  • mv fichier.txt nouveau_nom.txt → Renommer un fichier

🔧 Divers

  • tail -n 10 fichier.txt → Afficher les 10 dernières lignes
  • echo $PATH → Afficher les chemins système
